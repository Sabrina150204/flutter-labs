 DailyWin - Трекер Привычек

"DailyWin" — это простое мобильное приложение для отслеживания ежедневных привычек. Оно помогает пользователям формировать полезные рутины и видеть свой прогресс в виде наглядной статистики

 Основные функции

*Просмотр списка привычек: Главный экран отображает все привычки и их статус выполнения на сегодня.
*Отметка выполнения: Возможность отметить привычку как выполненную за сегодняшний день.
*Добавление новых привычек: Создание новых привычек с указанием названия и описания.
*Просмотр статистики: Визуализация прогресса по каждой привычке за последние 7 дней.
*Мотивационные цитаты: Показ случайной мотивационной цитаты, загружаемой из публичного API.

Структура экранов и навигация

Навигация в приложении осуществляется через нижнюю панель вкладок (Bottom Navigation Bar) между двумя основными экранами: "Мои Привычки" и "Статистика". Экран "Добавление привычки" открывается как модальное окно или отдельный экран с кнопки на главном экране.
